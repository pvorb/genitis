Gorn (Version 2.1)

Seit August 2010 wird diese Website vom schlichten Design Gorn geziert.

Der Name stammt von der gleichnamigen Figur aus der deutschen Computerspiele-Serie Gothic.

Technische Umsetzung

Konkret bestehen drei Layouts jeder Seite. (Zum Vergrößern klicken)

Eine breite Version für Bildschirme mit mindestens 980 Pixeln Breite

Eine schmale Version für Bildschirme zwischen 620 und 980 Pixeln Breite

Eine Version für kleine tragbare Geräte wie Smartphones oder PDAs

Zudem wird für die Darstellung großer Bilder oder sonstiger Inhalte – auch auf dieser Seite – die ColorBox von Jack Moore verwendet, welche unter der MIT-Lizenz verfügbar ist.

Gorn ist das erste Design, das die CSS 3-Technik media queries einsetzt, sodass für unterschiedliche Bildschirmgrößen unterschiedliche Anordnungen der einzelnen Elemente entstehen.

Die Idee dafür habe ich aus einem Artikel des Online-Magazins A List Apart, den ich nur empfehlen kann.

Dafür sind keinerlei Änderungen am Inhalt nötig, alles wird per CSS erledigt. Allerdings ist – wie so oft – der Internet Explorer der Spielverderber, denn er versteht CSS 3 noch nicht (bis einschließlich Version 8). Mit Hilfe eines JavaScripts kann man ihm jedoch auf die Sprünge helfen.

Einflüsse

Das Takeshi-Theme von Anmut und Demut hat das Design maßgeblich beeinflusst. Vor allem die Ähnlichkeit zwischen den schwarzen Leisten am oberen Bildschirmrand ist unverkennbar. Jedoch gab es auch schon bei Alma diesen schwarzen Balken, nur etwas dicker.

Die HTML- und CSS-Grundsätze stammen jedoch eher von Meiert.com.
