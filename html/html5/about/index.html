Über Genitis

Genitis ist die persönliche Website von Paul Vorbach.

Die Website besteht seit März 2008. Sie wurde als Weblog unter der Domain paulvorbach.de gestartet, läuft aber bereits seit Ende 2008 unter der Domain Genitis.org.

Versionen

Seit die Website unter Genitis.org erreichbar ist, hat es zwei grundlegende sowie eine technische Neugestaltung gegeben.

  • 1 Alma
  • 2 Gorn
    • 2.1 Gorn XHTML
    • 2.2 Gorn HTML5

Genitis verwendet das PHP-Framework Yuki.

Genitis läuft auf einem Server der Firma Sysprovide.

Alle Inhalte von Genitis.org stehen auf GitHub zur freien Verfügung.
