Voraussetzungen – Yuki-Tutorial

Sie benötigen zum Betrieb von Yuki einen Apache Web-Server, auf dem PHP eingerichtet ist. Dabei ist es egal, ob Sie Linux, Windows oder Mac als Betriebssystem verwenden.

Ein fertig konfiguriertes Programmpaket ist beispielsweise das beliebte XAMPP (Win, Mac, Linux) oder auch Wamp für Windows.

Verwenden Sie Webspace eines Providers, benötigen Sie einen FTP-Client, mit dem Sie die Dateien auf den entfernten Server laden können. Es empfiehlt sich, auch in diesem Fall einen lokalen Webserver aufzusetzen, um Änderungen zu testen, bevor man Sie veröffentlicht.

Außerdem wird die Verwendung eines Versionierungssystems wie beispielsweise Git empfohlen, da Änderungen notfalls rückgängig gemacht werden können.
